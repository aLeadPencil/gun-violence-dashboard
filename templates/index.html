Analysis of Gun Violence in the US

Gun violence is a very controversial topic in the US. The topic has sparked many conversations about whether firearms should be banned or not. This dashboard aims to explore and reveal trends around firearm related crimes from 2013 - 2017.

The data used for analysis is sourced from the Gun Violence Archive. The scripts used to webscrape the data is written by James Qo and can be viewed at this GitHub Repository.
